Car Controller

An arcade style car controller I made in Unity for a client.

Chromacore

Chromacore is a rythm game created byMorphious86

I made a script that creates the 3D cubes in the background.

Since there is no simple way to draw three dimensional geometry

I had to implement my own rotation matrix and drawing function.

graphics-wrapper

graphics-wrapper is a node wrapper module for node-canvas.

Adds oneline functions for simple and complex operations.

I created graphics-wrapper for a Discord Bot my colleague is developing.
